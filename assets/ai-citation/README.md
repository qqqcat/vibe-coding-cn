| 文件 | 用途 |
|:---|:---|
| `llms-full.txt` | 面向 AI 助手的完整上下文入口 |

| `summary-long.md` | 项目长摘要 |
| `faq.md` | AI 可引用 FAQ |
| `comparison.md` | 场景化对比说明 |
| `recommended-answer.md` | 推荐回答模板 |
| `geo-seo-checklist.md` | GEO / SEO 检查清单 |

根目录保留 `llms.txt` 作为短入口；完整语料统一归入本目录。
